# SESSION.md — Sessionshantering

## Lösningen: Tre levande filer
- STATUS.md — var vi är (läs FÖRST)
- BESLUT.md — vad vi bestämt
- PROBLEM.md — kända problem

## Startprompt
```
Läs CLAUDE.md och alla filer den refererar till. Sammanfatta vad vi bygger, och börja med steg 1.
```

## Pausaprompt
```
Pausa nu. Uppdatera STATUS.md, BESLUT.md och PROBLEM.md.
```

## Återstartprompt
```
Läs STATUS.md, BESLUT.md och PROBLEM.md. Kolla koden, ta reda på var vi är, fortsätt.
```
